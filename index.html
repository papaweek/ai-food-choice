5พื้นที่ครอบคลุม
4เกณฑ์คะแนน
–อัปเดตล่าสุด
ข้อมูลพร้อมใช้งาน — คลิก Scrape เพื่ออัปเดต
⚙️ วิธีตั้งค่า — 3 ขั้นตอน
Step 1 — JSONBin
1. ไปที่ jsonbin.io → Sign Up
2. กด New Bin
3. วาง {"restaurants":[]}
4. กด Create → Copy BIN ID
Step 2 — แก้โค้ด Dashboard
เปิด index.html แก้ 2 บรรทัดนี้:
const JSONBIN_BIN_ID = 'ใส่ BIN ID';
const JSONBIN_API_KEY = 'ใส่ Master Key'; แล้ว deploy ขึ้น Netlify ใหม่
Step 3 — ตั้งค่า n8n
เพิ่ม HTTP Request node ท้าย workflow:
Method: PUT
URL: https://api.jsonbin.io/v3/b/
      [BIN_ID]
Header: X-Master-Key: [KEY]
Body: ข้อมูลร้านอาหาร JSON
✅ หลังตั้งค่าครบ กด Scrape อีกครั้ง — Dashboard จะดึงข้อมูลล่าสุดจาก JSONBin มาแสดงทันที
🔎 n8n Response (raw) — ช่วย debug ว่า workflow ส่งอะไรกลับมา

    

      💡 Dashboard คาดหวัง response เป็น JSON array ของร้านอาหาร เช่น:

      [{"title":"ชื่อร้าน","totalScore":4.8,"reviewsCount":500,"price":"฿200-400","area":"อารีย์",...}]

      หรือ {"restaurants":[...]}  | 
      {"data":[...]}
    

  







  

    

      
🔧 CORS ยังไม่ผ่าน — ทำตามขั้นตอนนี้ใน n8n

      
    

    

      

        
ขั้นตอนที่ 1 — ตั้งค่า Webhook Node

        

          1. เปิด Workflow → คลิก Webhook node

          2. HTTP Method → POST

          3. Response Mode → Last Node
        

      

      

        
ขั้นตอนที่ 2 — Respond to Webhook Node

        

          เพิ่ม Respond to Webhook node ท้าย workflow

          ตั้ง Response Headers:

          Access-Control-Allow-Origin: *
Access-Control-Allow-Methods: POST, GET, OPTIONS
Access-Control-Allow-Headers: Content-Type
        

      

    

    

      ✅ หลังแก้แล้ว กด Save → Publish ใน n8n แล้วกด Scrape อีกครั้ง

    Scoring Criteria
  

  

    

      
⭐

      
Google Rating

      
30 pts

      
คะแนนจาก Google Maps (3.0–5.0) → 0–30 คะแนน ถ่วงน้ำหนักด้วยความน่าเชื่อถือ

    

    

      
💬

      
Review Popularity

      
20 pts

      
จำนวนรีวิว (log scale) → 0–20 คะแนน ยิ่งรีวิวมากยิ่งน่าเชื่อถือ

    

    

      
💰

      
Price Fit

      
25 pts

      
ราคาต่อหัวในช่วง 200–800 บาท → 25 คะแนน เหมาะสมสำหรับกลุ่ม 8–12 คน

    

    

      
👥

      
Group Suitability

      
25 pts

      
ถูก scrape จาก "group/large group/15 people" keywords → 22–25 คะแนน
